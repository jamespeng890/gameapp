🔧 编译方式
================

【方式1：使用构建脚本（推荐）】

Windows:
  build.bat debug      → 构建Debug版本
  build.bat release    → 构建Release版本（含混淆）
  build.bat install    → 安装到设备
  build.bat run        → 安装并运行

Linux/Mac:
  ./build.sh debug
  ./build.sh release
  ./build.sh install
  ./build.sh run

【方式2：使用Gradle命令】
  gradlew assembleDebug     → 生成Debug APK
  gradlew assembleRelease   → 生成Release APK
  gradlew installDebug      → 安装Debug版本
  adb install app-debug.apk → 手动安装

【方式3：使用Android Studio】
  1. File → Open → 选择gameapp文件夹
  2. 等待Gradle同步
  3. Build → Make Project
  4. Run → Run 'app'
  5. 选择设备或模拟器

🎯 算法亮点
================

【匹配检测】
- 时间复杂度：O(n×m) - 按行列扫描
- 水平和竖直同时检测
- 一遍完成所有匹配查询

【棋盘填充】
- 时间复杂度：O(n×m)
- 先下移再填充，高效处理
- 支持链式反应递归处理

【粒子管理】
- 对象池复用，无GC压力
- 硬上限500个活跃粒子
- 每帧O(n)更新和绘制

【缓动动画】
- 7种缓动函数（线性、二次、回弹等）
- 平滑的视觉效果
- 可组合多个效果
